Telethon Setup Required

The dashboard reads job history from your Telegram channel via Telethon. A one-time authentication is needed before it can work.

Steps:

  1. Get your api_id and api_hash from my.telegram.org → API Development Tools
  2. Fill them into config.yaml under telegram.api_id and telegram.api_hash
  3. Run: python auth_telethon.py
  4. Restart: python run.py
